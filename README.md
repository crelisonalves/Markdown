# Texto básico
**Negrito** *itálico* ***ambos***

~~~
   **Negrito**
   *itálico*
   ***negrito e itálico***
~~~
Exibir caracteres especiais no Markdown use a barra invertida \ \*, \**, \***, etc.

~~~
\*
\**
\***
~~~
